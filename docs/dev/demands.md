# 睡意绵绵demo-20260608（已归档）
依托原版，新增两种床样式，分别对应弱于原版床的debuff与强于原版床的buff

## 床的制作基础构思
●弱于原版床-草席，初步设想可以通过割草获得，原版草在锄头割除时有概率（10%）掉落草,2个草合成一个草束,6个草束(3*2配方结构同门)->1个草席,然后使用原版工作台进行3*2做成草席，一次性，可度过夜晚，但获得虚弱debuff——1min

●强于原版床-羊驼床: 击杀羊驼获得羊驼毛皮，之后然后使用原版工作台，demo版先用合成表，无序，放置羊驼毛皮+剪刀——>产出羊驼毛（可以集成到后续版本中，但使用mod内处理模块，能得到更多物品，如羊驼皮），之后在原版床的合成配方基础上，替换羊毛为羊驼毛，可重复使用，度过夜晚后，得到力量1、幸运1——3min


基于以上设定，需要以下产出
## 新增物品

草束——用锄头破坏草获得草后，用草合成获得.需要绘制物品材质

羊驼毛皮——杀害羊驼后概率掉落1-2张,需要绘制物品材质

羊驼毛——工作台,羊驼皮毛+剪刀(无序，不消耗剪刀但消耗耐久)合成,需要绘制物品材质

草席——可放置占地1*2，与床相同；合成方式2*3与门相同合成,需要绘制物品和方块材质

羊驼床——基于原板床修改材质蓬松一些，可放置占地1*2，合成表在原版床基础上羊毛替换为羊驼毛.需要绘制物品和方块材质


## 新增逻辑:

手持锄头左键破坏草有概率掉落草

在草席上度过夜晚,起床后获得虚弱buff 1min

在羊驼床上度过夜晚,起床后获得 buff 力量1、幸运1——3min
